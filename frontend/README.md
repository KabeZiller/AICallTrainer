# AI Call Trainer - Frontend

React + TypeScript + Vite frontend for the AI Call Trainer platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
